Data Analysis with R and Python

Course Project

As part of this course, there will be group projects that run over whole semester. Each group consists of six (or five in some cases) students, and each student is in two groups.

The job of each group is to find an "interesting" dataset and present an analysis. Some sources have been shared in Google Classroom to help with this.

Please use Google Classroom to discuss projects, and email me to schedule an online meeting with me if necessary.
